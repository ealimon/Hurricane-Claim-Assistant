That isn’t a relevant question.” Note: In demo mode, answers are sample, not grounded to the live page. -------------------------------------------------------------------------------- Optional: Deploy GitHub Pages (share a live link) -------------------------------------------------------------------------------- 1) Commit and push this repo to GitHub 2) Settings → Pages → Build and deployment → Source: “GitHub Actions” 3) The included workflow (.github/workflows/deploy-pages.yml) will publish /public 4) Share the URL: https://.github.io/hurricane-claim-assistant/public/assistant.html -------------------------------------------------------------------------------- Optional: Real answers (Cloudflare Worker + OpenAI) -------------------------------------------------------------------------------- The backend fetches and caches your landing page content, extracts relevant snippets, and asks OpenAI to answer strictly from that text. It also enforces the relevance gate server-side so out-of-scope questions get your exact message. What you need - Cloudflare account + Wrangler CLI - OpenAI API key Steps 1) Edit backend/wrangler.toml: - Set name to something unique 2) Deploy the Worker: - npm install -g wrangler - cd backend - wrangler login - wrangler deploy - In Cloudflare dashboard → Worker → Settings → Variables, add: - OPENAI_API_KEY = your key - PAGE_URL = https://www.yourpropertyadvocate.com/lp/hurricane-damage-ppc_lp/ 3) Copy the Worker URL (e.g., https://hca-demo..workers.dev) 4) Update public/assistant.html: - Set DEMO_MODE = false - Set BACKEND_URL to https://hca-demo..workers.dev/api/assistant-answer 5) Refresh and test Notes - The Worker uses simple keyword retrieval across the page text for reliability and speed. - It adds a short legal-safe instruction and will refuse anything not supported by the page. - You can upgrade retrieval later (embeddings/vector DB) if needed. -------------------------------------------------------------------------------- License -------------------------------------------------------------------------------- MIT — see LICENSE. ================================================================================ public/assistant.html ================================================================================ Hurricane Claim Assistant

================================================================================ backend/cloudflare-worker.js ================================================================================ /** * Cloudflare Worker — Simple grounded Q&A for the landing page * - Fetches and caches page text for 24h * - Simple keyword retrieval over chunks * - Calls OpenAI to answer strictly from context * * Env Vars (set in Cloudflare dashboard): * - OPENAI_API_KEY * - PAGE_URL (default in code if not set) */ export default { async fetch(request, env, ctx) { const url = new URL(request.url); // CORS (allow embedding domain if needed; * for demo) if (request.method === "OPTIONS") { return new Response(null, { status: 204, headers: corsHeaders(), }); } if (url.pathname === "/api/assistant-answer" && request.method === "POST") { try { const body = await request.json(); const question = (body?.question || "").trim(); if (!question) return json({ error: "Missing question" }, 400); // Relevance guard (server-side) const relevant = isRelevant(question); if (!relevant) { return json({ answer: "I’m sorry. That isn’t a relevant question." }); } const pageUrl = env.PAGE_URL || "https://www.yourpropertyadvocate.com/lp/hurricane-damage-ppc_lp/"; const pageText = await getPageText(pageUrl, request, ctx); // Retrieve relevant snippets const snippets = retrieveSnippets(pageText, question, 6, 900); const context = [ "Source: Hurricane damage landing page content.", "Phone: 813-445-5780. Office: 5331 Primrose Lake Circle, Suite 212, Tampa, FL 33647.", "Key themes: Free consultation, zero upfront cost, focus on property damage, Tampa Bay area (Hillsborough, Pinellas, Pasco), recent results include roof replacements and interior water damage payments." ].concat(snippets).join("\n\n---\n\n"); const prompt = buildPrompt(question, context); const answer = await callOpenAI(prompt, env.OPENAI_API_KEY); return new Response(JSON.stringify({ answer }), { status: 200, headers: { "content-type": "application/json", ...corsHeaders() }, }); } catch (err) { return json({ error: "Server error" }, 500); } } return new Response("OK", { status: 200, headers: corsHeaders() }); }, }; function corsHeaders() { return { "access-control-allow-origin": "*", "access-control-allow-methods": "GET, POST, OPTIONS", "access-control-allow-headers": "content-type, authorization", }; } function json(obj, status = 200) { return new Response(JSON.stringify(obj), { status, headers: { "content-type": "application/json", ...corsHeaders() }, }); } // Basic relevance keywords const RELEVANCE_KEYWORDS = [ "hurricane","storm","wind","roof","leak","damage","water","flood","property","property damage", "insurance","claim","claims","denied","underpaid","low offer","adjuster","attorney","law","law firm", "consultation","free consultation","no fees","zero upfront","contingency", "tampa","pinellas","hillsborough","pasco","tampa bay","florida", "karkatselos","your property advocate","results","successes", "office","address","location","contact","call","phone","813","map" ]; function isRelevant(q) { const qs = q.toLowerCase(); return RELEVANCE_KEYWORDS.some(kw => qs.includes(kw)); } // Fetch and cache page text for 24h async function getPageText(pageUrl, request, ctx) { const cache = caches.default; const cacheKey = new Request("https://cache-key/" + btoa(pageUrl), { method: "GET" }); const cached = await cache.match(cacheKey); if (cached) return cached.text(); const res = await fetch(pageUrl, { headers: { "user-agent": "AssistantBot/1.0" } }); const html = await res.text(); const text = extractText(html); ctx.waitUntil(cache.put(cacheKey, new Response(text, { headers: { "cache-control": "max-age=86400" } }))); return text; } // Strip tags and boilerplate function extractText(html) { // remove scripts and styles html = html.replace(//gi, " ").replace(//gi, " "); // replace breaks with newlines html = html.replace(//gi, "\n").replace(/<\/p>/gi, "\n").replace(/<\/h[1-6]>/gi, "\n"); // strip tags const text = html.replace(/<[^>]+>/g, " ").replace(/\s+/g, " ").trim(); return text; } // Chunk text and score by keyword overlap function retrieveSnippets(text, question, maxSnippets = 6, chunkSize = 900) { const chunks = chunk(text, chunkSize, 200); const qTokens = tokenize(question); const scored = chunks.map((c, i) => ({ i, score: scoreChunk(c, qTokens), c })); scored.sort((a, b) => b.score - a.score); return scored.slice(0, maxSnippets).map(s => s.c); } function chunk(str, size, overlap = 0) { const out = []; for (let i = 0; i < str.length; i += size - overlap) out.push(str.slice(i, i + size)); return out; } function tokenize(s) { return s.toLowerCase().split(/[^a-z0-9]+/).filter(Boolean); } function scoreChunk(c, qTokens) { const lc = c.toLowerCase(); let score = 0; for (const t of qTokens) { if (lc.includes(t)) score += 1; } return score; } function buildPrompt(question, context) { return [ { role: "system", content: "You are the Hurricane Claim Assistant for a hurricane-damage landing page. " + "Answer concisely (2–5 sentences). Use ONLY the provided context. " + "If the answer isn’t supported by the context, reply exactly: \"I’m sorry. That isn’t a relevant question.\" " + "Include a gentle note: \"General information—not legal advice.\" " + "If appropriate, you may suggest calling 813-445-5780 to schedule a free consultation." }, { role: "user", content: "Context:\n" + context + "\n\nQuestion:\n" + question } ]; } async function callOpenAI(messages, apiKey) { const res = await fetch("https://api.openai.com/v1/chat/completions", { method: "POST", headers: { "content-type": "application/json", "authorization": `Bearer ${apiKey}` }, body: JSON.stringify({ model: "gpt-4o-mini", messages, temperature: 0.2, max_tokens: 220, }), }); if (!res.ok) { const err = await res.text(); throw new Error("OpenAI error: " + err); } const data = await res.json(); const answer = data?.choices?.[0]?.message?.content?.trim() || ""; return answer || "I’m sorry. That isn’t a relevant question."; } ================================================================================ backend/wrangler.toml ================================================================================ name = "hca-demo" main = "cloudflare-worker.js" compatibility_date = "2024-11-20" # Set environment variables in the Cloudflare Dashboard after deploy: # - OPENAI_API_KEY # - PAGE_URL # For local testing: # wrangler dev ================================================================================ .github/workflows/deploy-pages.yml ================================================================================ name: Deploy GitHub Pages on: push: branches: ["main"] workflow_dispatch: {} permissions: contents: read pages: write id-token: write concurrency: group: "pages" cancel-in-progress: true jobs: build: runs-on: ubuntu-latest steps: - name: Checkout uses: actions/checkout@v4 - name: Upload artifact uses: actions/upload-pages-artifact@v3 with: path: ./public deploy: needs: build runs-on: ubuntu-latest environment: name: github-pages url: ${{ steps.deployment.outputs.page_url }} steps: - name: Deploy to GitHub Pages id: deployment uses: actions/deploy-pages@v4 ================================================================================ LICENSE ================================================================================ MIT License Copyright (c) 2025 Permission is hereby granted, free of charge, to any person obtaining a copy of this software and associated documentation files (the "Software"), to deal in the Software without restriction, including without limitation the rights to use, copy, modify, merge, publish, distribute, sublicense, and/or sell copies of the Software, and to permit persons to whom the Software is furnished to do so, subject to the following conditions: The above copyright notice and this permission notice shall be included in all copies or substantial portions of the Software.
